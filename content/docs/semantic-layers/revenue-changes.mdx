export const caption = "A list of Revenue Changes"

## Semantic Contract
### Definitions

| Column | Description |
|--------|-------------|
| period_id | Factual reference to the period (links to [periods](semantic-layer/periods)). |
| revenue_type_id | Type of revenue change (links to [revenue_types](semantic-layer/revenue_types), e.g., New Booking, Upsell). |
| value_type_id | Type of value measured (links to [value_types](semantic-layer/value_types), e.g., actuals, budget). |
| nr_of_customers | Volume variable with number of customers contributing to this revenue change. |
| value_per_customer | Pricing variable, with revenue per customer for this change. |
