## Dependencies

### Internal

- **[Business Units](./business-units)**: required when `is_segmented=true`.

### External

- None.

## Integration

This dimension defines and returns the SQL required to create the `gl_accounts` table. Table creation and lifecycle management are handled by the database [Methods](/components/database/methods) and [Schema Manager](/components/database/schema-manager).

When `is_segmented=true`, the schema extends to reference the [Business Units](./business-units) dimension.
When `false`, the dimension remains fully self-contained.
