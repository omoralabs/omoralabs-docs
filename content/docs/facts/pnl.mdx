## Facts
### Definitions

| Column | Description |
|--------|-------------|
| date | Date in YYYY-MM-DD format, reference to [Dates](semantic-layer/dates). |
| gl_account_id | Which GL Account (links to [GL Accounts](semantic-layer/gl-accounts)). |
| value_type_id | Type of value measured (links to [Value Types](semantic-layer/value-types), e.g., actuals, budget). |
| amount | Fact registered in a currency unit. |
