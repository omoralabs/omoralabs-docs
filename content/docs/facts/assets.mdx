## Facts
### Definitions

| Column | Description |
|--------|-------------|
| date | Factual reference to the date (links to [Dates](semantic-layer/dates)). |
| asset_id | Asset measured (links to [Assets Providers](semantic-layer/asset-providers), e.g. Stripe, Mercury, HSBC). |
| amount | Fact registered in a currency unit. |
